# 🛒 M Ecommerce Sales Dashboard — Power BI

An interactive ecommerce sales dashboard built in Power BI to analyze sales performance across states, categories, and payment modes.

## 🔧 What I Did

**1. Data Cleaning (Excel)**
- Fixed inconsistent date formats
- Handled null/missing values
- Exported cleaned file as CSV

**2. Data Import**
- Imported the CSV into Power BI Desktop

**3. Dashboard Building**

| Feature | Details |
|---|---|
| KPI Cards | Amount: 438K · Profit: 37K · Quantity: 6K · AOV: 121K |
| AOV Table | Calculated Average Order Value by category |
| Bar Chart | Sales amount by State |
| Donut Charts | Quantity by Category · by Payment Mode · AOV by Category |
| Column Chart | Profit by Month (Jan–Dec) |
| Bar Chart | Profit by Sub-Category |
| Filters | Quarter slicer (Qtr 1–4) + State dropdown |
